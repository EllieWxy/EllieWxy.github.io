js转json

将字符串转json的三种方法
  • var json = JSON.parse(str);
  • var json = eval(“(“ + str + “)”);
  • var json = (new Function(“return “ + str))();
JSON.parse注意事项

例如字符串var str = '{"name":"小明","age":18}';中，是单括号套双括号，以下两种方式都会报错。

![image-20200517214637458](/Users/EllieWxy/Library/Application Support/typora-user-images/image-20200517214637458.png)

总结：

前面的name/age一定要用""标起来，如果值是字符串的话也必须用双引号，外面必须用单引号

JSON.stringify

此时一定要保证转化的是json格式，否则就会报错。

  • JSON.stringify只能转化为能成为字符串的

  • undefined 值、函数或者XML值会被忽略

  • 如果你的数组当中含有 undefined值，函数或XML值，该数组中的这些值将会被当成 null
